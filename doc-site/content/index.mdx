# Lazybones

**Lazybones** is a reusable edge/cloud platform for extension-driven, local-first
workspace software — a quiet, capable control surface for inspecting workspace
state, routing work through extensions, and reasoning about capability-scoped
data.

This site renders the **published** documentation. Authoring happens in
[`../docs`](https://github.com/NubeIO/lb/tree/main/docs); what you read here is
`docs/public` (what shipped) and `docs/vision` (the north star). Working
material — `scope/` and `sessions/` — is not published.
