# Lazybones

**Lazybones** is a reusable edge/cloud platform for extension-driven, local-first
workspace software — a quiet, capable control surface for inspecting workspace
state, routing work through extensions, and reasoning about capability-scoped
data.

This site renders the **published** documentation. The shipped docs live in
[`doc-site/content/public`](https://github.com/NubeIO/lb/tree/main/doc-site/content/public) (what shipped,
authored here as MDX); `vision/` is the north star. Working material — `scope/` and `sessions/` — is not
published.
