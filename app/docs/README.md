# App docs — pointer

Authored documentation lives in the central tree (the project convention —
scope → sessions → public):

- The ask: `docs/scope/app/` (start at its `README.md`)
- Working logs: `docs/sessions/app/`
- Shipped truth: `docs/public/app/app.md`

This folder intentionally holds nothing else, mirroring how extension docs point back
to the central tree unless the component is 100% self-contained.
