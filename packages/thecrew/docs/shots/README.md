# docs/shots — the screenshot test artifacts

Every phase ends with screenshots of both demos checked in here
(`look-scope.md` §the bar). Naming: `phase<N>-<demo>-<flat|3d>.png`,
e.g. `phase1-ahu-flat.png`. A shot passes the four look-scope criteria:
no defaults, status legible at 3 m, calm motion (judged live), poster/HUD quality.
