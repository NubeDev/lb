| 2026-07-01 | reminders | the channel `/reminders` table rendered **ONE JSON-blob row**, not N reminders — so a per-row control's `${id}` bound nothing (the interactive list was inert). | 2026-07-01 | reminders | **[open, pre-existing]** every `reminder.fire` (and the scheduled reactor firing) **denies** for a **dev-login / token-only** principal, even though its token provably carries the fired action's cap — so the channel `/reminders` "Run now" control is inert over the real gateway. | 2026-07-01 | store | `pnpm test:gateway` flaked `not a member of any workspace` from `signInReal` (varies run-to-run) — many fresh `nextWs()` workspaces log into the ONE spawned gateway concurrently.
