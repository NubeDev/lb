# Nav — user-/team-authored navigation (public)

Status: **TODO** — placeholder. Filled in when the nav builder ships.

Scope: [`scope/nav/nav-builder-scope.md`](../../scope/nav/nav-builder-scope.md).

The nav builder lets an admin compose a navigation menu (`nav` asset) whose entries link to dashboard
pages, system surfaces, extension pages, or dynamic tag-groups, assign it to teams, and have each member
see it resolved to their effective menu. The menu is a **lens over existing access, never a grant path**.

This page will document the shipped truth (verbs, caps, records, the resolver, the builder UI) once the
feature lands — promoted from the scope + session docs per `ABOUT-DOCS.md`.
