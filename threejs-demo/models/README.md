# models

Drop a real gas-station model here as `station.glb` and reload — the demo will use it
instead of the built-in procedural station.

Suggested free model: **Small Red Gas Station** (SmashedBones) on Sketchfab — download
the glTF/GLB and rename to `station.glb`.

The demo logs the model's named parts to the browser console so you can line up the
hotspots in `../src/layout.js`.
